# 高中公民與社會｜行政法基本概念 102 頁互動式 HTML 簡報

本專案係專為高中學生（108 課綱公民與社會科範疇）量身打造之高質感、高度互動性 HTML5 教學簡報系統。包含 102 頁精心規劃之法理教學內容、素養導向選擇題、配合題與簡答題，並配置流暢微光澤（Glassmorphism） UI 與動態切換動畫。

---

## 🌟 專案亮點與特色

1. **102 頁完整大綱與邏輯分層**：
   - 劃分為 8 大主題模組，從行政本質、依法行政、一般法律原則、行政行為態樣、行政制裁、行政執行至行政救濟與國家責任全盤涵蓋。
2. **豐富素養導向題型**：
   - **素養選擇題**：附情境故事（如疫情管制、酒駕併罰、學校處分），實時驗證並附法理解析。
   - **素養配合題**：互動式卡片點擊連線配對（如層次化法律保留配對、公私法行為配對）。
   - **素養簡答題**：開放式思考引導，附標竿參考解答與「採分關鍵字」自我對照。
3. **動態動畫與視覺美學**：
   - 提供淡入滑動 (Slide Fade)、鏡頭縮放 (Zoom)、3D 翻轉 (3D Flip)、動態彈跳 (Bounce) 等多種動畫切換效果。
   - 卡片交錯飛入 (Staggered Animations)、微光澤玻璃質感深色主題。
4. **流暢操作體驗與快捷鍵**：
   - 支持鍵盤方向鍵、目錄搜尋抽屜、講師重點補充抽屜、快速跳頁與全螢幕播放。

## ⌨ 簡報快捷鍵指南

- `→` / `Space` / `PageDown`：下一頁
- `←` / `PageUp`：上一頁
- `Home` / `End`：跳至第一頁 / 最後一頁
- `M`：開啟 / 關閉 102 頁大綱目錄與關鍵字搜尋抽屜
- `N`：開啟 / 关閉 講師重點解說抽屜
- `F`：切換全螢幕模式
- `?`：開啟快捷鍵說明面板
