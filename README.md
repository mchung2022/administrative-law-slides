# 高中公民與社會｜行政法基本概念 500 頁旗艦版 互動教學簡報

本專案係專為高中學生（108 課綱公民與社會科範疇）量身打造之高質感、高度互動性 500 頁 HTML5 旗艦級教學簡報系統。整合 80+ 新聞真實案例、憲判字/大法官解釋典範，並配置 50 題素養選擇題、30 題配合題與 30 題簡答題。

---

## 🌟 專案亮點與特色

1. **500 頁完整大綱與 10 大主題模組**：
   - 涵蓋行政概論、依法行政（釋字443）、比例原則與平等原則、信賴保護與裁量控制、行政處分全解、行政契約與行政立法、行政罰法與真實裁罰、行政執行法與即時強制、訴願與行政訴訟三類型、國家賠償與損失補償大會考。
2. **80+ 時事新聞與真實法理案例**：
   - 包含疫情隔離與戴口罩裁罰、酒駕刑事與行政併罰、台鐵公司化、年金改革信賴保護、山崩路段即時強制、路面坑洞國賠、違建拆除與代履行等。
3. **豐富素養導向三大題型**：
   - **素養選擇題 (50 題)**：附情境故事，實時驗證並附法理解析。
   - **素養配合題 (30 題)**：雙欄互動式卡片點擊連線配對與驗證。
   - **素養簡答題 (30 題)**：開放式思考引導，附參考解答與「採分關鍵字」清單。
4. **多重切換動畫與高效導覽**：
   - 提供淡入滑動 (Slide Fade)、鏡頭縮放 (Zoom)、3D 翻轉 (3D Flip)、動態彈跳 (Bounce) 等動畫效果。
   - 支援 500 頁快速關鍵字搜尋過濾抽屜、講師補充抽屜與全螢幕播放。
